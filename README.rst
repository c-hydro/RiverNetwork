RiverNetwork
=============================
**RiverNetwork** è una classe Matlab che ricostruisce la topologia di un reticolo idrografico a partire dalle mappe delle idroderivate e permette alcune operazioni su di esso, tra cui la corrispondenza ramo-ramo tra due rappresentazioni dello stesso reticolo.

Richiede i toolbox:
- Statistics
- Mapping

Guida dettagliata nel file **Manuale_RiverNetwork.pdf** o **Manuale_RiverNetwork.rst**
